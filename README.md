# 🎁 크리스마스 선물 실패 확률 계산기

단일 `index.html` 파일만 열면 동작합니다. Tailwind CDN + 순수 JS 구성.

## 계산식 요약
- 기본 성공률 70%에서 아래 항목을 차감해 최종 성공률을 계산하고 `5~95%`로 클램프.
- 관계 리스크: `연인 12`, `썸 12×1.4`, `부부 12×0.7`, `회사/지인 6`.
- 예산 패널티 (관계별):
  - 연인: `low +12`, `mid +4`, `high 0`, `premium -2`
  - 썸: `low +10`, `mid +4`, `high +2`, `premium +8`
  - 부부: `low +6`, `mid 0`, `high -2`, `premium +2`
  - 회사/지인: `low 0`, `mid +2`, `high +6`, `premium +10`
- 선물 유형 리스크: `실용 4`, `감성 8`, `패션 14`, `드립 16`, `경험 2`.
- 성향 매칭 패널티: `matchPenalty[giftType][persona]` 매트릭스 사용.
- 힌트 패널티: `직접 0`, `언급 2`, `눈치 10`, `그냥 느낌 20`.  
  성향이 `잘 모르겠음`이면 추가 +15.
- 실패률 기반 부가 요소:
  - `재앙 레벨`: 80%↑ 🧊, 60%↑ 😐, 40%↑ 😅, 그 외 🎄.
  - `싸늘해지는 속도`: 실패율 65%↑ 즉시, 45%↑ 하루 뒤, 25%↑ 며칠 뒤, 그 미만은 따뜻.
- 공유 문구: `“내 크리스마스 선물, 실패 확률 {fail}% 나옴. 왜 벌써 싸늘해졌는지 이해 완료.”`

## 파일
- `index.html`: 인터랙티브 계산기 UI + 로직 + 카피.
- `questions.json`: 질문/옵션 정의 (외부 시스템 연동 시 참고용).
